# Changelog

全てのリリースや大きな変更はこのファイルに記録してください。上から新しい変更履歴を追加します。

## [0.1.0] - 2026-01-31
- 在庫一覧のモーダルから在庫数を直感的に増減・確定できる「在庫操作」機能を追加。
- 左ペインの用途ツリーと中央テーブルのフィルタを用途ごとに同期し、未設定表示・棚番ソートを強化。
- Web UI の在庫一覧行をクリックすると、棚番／メーカー／在庫・発注点／最新トランザクションなどの詳細を確認可能に。
- 在庫変更を `/inventory/inline-adjust` エンドポイントでトランザクション管理し、API から行・ステータス・差分・最終取引を即時更新。
- `ZoneInfo("Asia/Tokyo")` で生成するタイムスタンプに統一し、表示と記録を日本時間に整合。
- `tzdata` と `python-multipart` を含む依存を `requirements.txt` に追加し、FastAPI でのフォーム・タイムゾーン処理を正常化。
